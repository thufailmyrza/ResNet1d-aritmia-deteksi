Arsitektur diawali dengan stem layer berupa Conv1D (kernel 15, stride 2), BatchNorm, ReLU, dan MaxPool ringan untuk mengekstraksi fitur morfologi dasar (QRS, P, T) sekaligus mereduksi resolusi temporal. Setelah itu terdapat empat residual stages dengan konfigurasi default [2, 2, 2, 2] blok (total 8 ResBlock). Setiap ResBlock terdiri dari dua Conv1D (kernel 7) dengan BatchNorm dan ReLU, dilengkapi shortcut connection untuk menjaga stabilitas gradien. Di dalam blok juga terdapat SEBlock yang melakukan channel-wise recalibration, sehingga model dapat memberi bobot lebih besar pada lead yang lebih informatif terhadap pola aritmia tertentu (misalnya lead inferior pada beberapa kasus ventricular event).

Dimensi kanal meningkat secara bertahap (64 → 128 → 256 → 512) sementara panjang sinyal menurun akibat stride 2 pada awal stage 2–4. Pendekatan ini memungkinkan ekstraksi fitur hierarkis: dari morfologi lokal (gelombang per beat) menuju pola ritme jangka pendek (irregularity, repetitive pattern seperti bigeminy/trigeminy). Setelah stage terakhir, diterapkan Multi-Scale Temporal Attention untuk menekankan segmen waktu yang paling relevan secara klinis, terutama pada window yang hanya mengandung sebagian episode aritmia.

Fitur akhir diringkas menggunakan Global Average Pooling, menghasilkan vektor 512 dimensi yang kemudian masuk ke classification head (Linear 512→256→11 dengan BatchNorm dan Dropout). Output berupa logits untuk 11 kelas aritmia, dan prediksi diperoleh melalui argmax. Untuk integrasi dengan sistem Holter, class index dikonversi menjadi flag berbasis bit-shift (1 << class_index) sehingga kompatibel dengan format arrhythmia.bin.

Secara keseluruhan, desain ini menggabungkan stabilitas pelatihan ResNet, sensitivitas kanal melalui SE attention, dan fokus temporal melalui attention 1D, sehingga sesuai untuk karakteristik sinyal ECG multi-lead yang bersifat non-stasioner dan sangat bergantung pada morfologi serta dinamika ritme.
